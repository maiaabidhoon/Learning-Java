* # Java:
    * Java is a high-level, class-based, and object-oriented programming language designed to have as few implementation dependencies as possible. It is famous for the "Write Once, Run Anywhere" (WORA) principle, meaning compiled Java code can run on all platforms that support Java without the need for recompilation. 
    * ## Key Features:
        * **Object-Oriented:** Everything is an object, which helps in organizing complex software into modular, reusable units.
        * **Platform Independent:** It uses bytecode instead of machine-specific code, making it highly portable.
        * **Automatic Memory Management:** Features a Garbage Collector that automatically manages memory by deleting unused objects.
        * **Secure:** Java applications run in a "sandbox" to prevent unauthorized access to system resources.
    * ## Common Uses:
        * **Mobile Apps:** It is the primary language for Android development.
        * **Enterprise Software:** Widely used by banks and large corporations for backend systems.
        * **Web Applications:** Powers server-side applications through frameworks like Spring and Hibernate.
        * **Big Data & AI:** Used in data processing engines and machine learning libraries.
    * ## Current Versions (as of March 2026):
        * **Latest Release:** Java 26 (released March 17, 2026).
        * **Long-Term Support (LTS):** The most recent LTS versions are Java 25, 21, 17, 11, and 8. 
    
    * ## Core Concept:
        * ### JVM (Java Virtual Machine): 
            * This is the core execution engine that translates Java bytecode into machine-specific instructions for the underlying hardware. It is an abstract machine and is platform-dependent (requires a specific implementation for each operating system), which is why Java is considered "write once, run anywhere" (WORA).
            
        * ### JRE (Java Runtime Environment):
            * This is a software package that provides the environment necessary to run compiled Java programs. It physically exists on a system and includes the JVM, core class libraries, and supporting files but contains no development tools. End-users who only need to run Java applications typically install the JRE.
            
        * ### DK (Java Development Kit): 
            * This is a comprehensive software development kit that provides all the tools and libraries needed to create, compile, and debug Java applications. The JDK is a superset of the JRE; installing the JDK automatically includes the JRE and additional development tools such as the javac compiler, jdb debugger, and jar archiver. Developers must install the JDK to write new Java programs.
